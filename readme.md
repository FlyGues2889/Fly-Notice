# <center> FlyNotice </center>

<center>
  基于 Tauri，使用 Material You 设计风格的通知展示程序
</center>



## 开发和调试

- 克隆项目代码

   ```
   git clone https://github.com/FlyGues2889/Fly-Notice.git

   cd fly-notice
   ```

   安装依赖：
   ```
   pnpm install
   ```


- 运行项目：

   ```
   pnpm dev
   ```

- 打包安装包：

   ```
   pnpm build
   ```
